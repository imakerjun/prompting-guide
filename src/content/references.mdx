# 참고 자료

이 강의에서 다룬 내용을 더 깊이 학습하거나, 실무에 바로 적용할 수 있는 신뢰할 수 있는 자료들을 모았습니다.
모든 자료는 AI 제공사의 공식 문서, 검증된 교육 플랫폼, 또는 학술 논문을 기반으로 선별했습니다.

---

## 공식 문서

이 강의의 원본 자료입니다. 강의에서 배운 원리를 원문으로 확인하고 싶을 때 참고하세요.

| 자료 | 설명 |
|------|------|
| [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) | **이 강의의 원본**. Claude 프롬프트 엔지니어링 공식 가이드 |
| [Anthropic Prompting Best Practices](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/claude-4-best-practices) | Claude 4 모델별 프롬프팅 모범 사례 종합 레퍼런스 |
| [OpenAI Prompt Engineering Guide](https://developers.openai.com/api/docs/guides/prompt-engineering) | OpenAI 공식 가이드. 6가지 전략을 체계적으로 정리 |
| [Google Prompt Design Strategies](https://ai.google.dev/gemini-api/docs/prompting-strategies) | Gemini용 프롬프트 설계 전략. Few-shot, CoT 등 |
| [Microsoft Prompt Engineering Techniques](https://learn.microsoft.com/en-us/azure/ai-services/openai/concepts/prompt-engineering) | Azure OpenAI용 프롬프트 엔지니어링 기법 |

> 여러 AI 서비스의 공식 문서를 비교해 보면, **명확성, 맥락 제공, 예시 활용** 같은 핵심 원칙은 모든 AI에 공통으로 적용된다는 것을 알 수 있습니다.
