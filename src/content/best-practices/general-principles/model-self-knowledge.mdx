# AI에게 정체성 설정하기

<OriginalDocLink href="https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/claude-prompting-best-practices#model-self-knowledge">공식문서에서 보기</OriginalDocLink>

"모델 자기 인식"이란 Claude가 자신의 역할과 정체성을 어떻게 이해하는가입니다. 직장인이 Claude에게 **"너는 우리 팀의 마케팅 어시스턴트야"**라고 정체성을 설정하면, 일관된 관점에서 답변합니다.

새 직원이 입사했을 때 "당신은 마케팅팀 소속이고, 우리 주 고객은 이런 사람들이야"라고 알려주는 것처럼, AI에게도 **누구인지, 어떤 맥락에서 일하는지** 알려주면 훨씬 맞춤화된 답변을 받을 수 있습니다.

<Callout emoji="💡">
정체성 설정은 단순히 API의 모델 문자열을 지정하는 것보다 훨씬 넓은 개념입니다. **AI가 어떤 조직에 소속되어 있고, 어떤 관점에서 답변해야 하는지**를 정의하는 것입니다. Claude.ai, ChatGPT 등 어떤 도구에서든 대화의 시작 부분에 정체성을 설정할 수 있습니다.
</Callout>

## 정체성 없이 vs 정체성 설정 후

<Tabs items={['❌ 정체성 없이', '✅ 정체성 설정']}>

```
마케팅 전략 아이디어 줘
```

Claude: 일반적이고 교과서적인 마케팅 전략 나열 (타겟이 불분명하고, 우리 회사 상황과 맞지 않는 답변)
  </Tabs.Tab>
  <Tabs.Tab>
```
너는 우리 회사의 마케팅팀 소속이야.
우리는 B2B SaaS 제품을 판매하고, 주 고객은 중소기업 IT 담당자야.
이 맥락에서 마케팅 전략 아이디어를 줘
```

Claude: B2B SaaS와 중소기업 IT 담당자에 특화된 전략 제안 (기술 블로그 마케팅, 무료 체험 전환 전략, IT 커뮤니티 타겟 광고 등)
  </Tabs.Tab>
</Tabs>
