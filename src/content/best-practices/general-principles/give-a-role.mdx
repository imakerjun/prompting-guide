## Claude.ai에서 역할 설정하기

<SlideOnly>
<div className="slide-text-center">
<p style={{marginBottom: '16px'}}>많은 분들이 Claude.ai를 쓰실 텐데, API는 필요 없습니다.<br/><strong>대화의 첫 메시지</strong>가 곧 시스템 프롬프트예요.</p>
<pre style={{textAlign: 'left', display: 'inline-block', padding: '16px 24px', background: '#f5f5f5', borderRadius: '8px', fontSize: '0.9em'}}>{`당신은 10년 경력의 마케팅 전문가입니다.
우리 회사는 B2B SaaS 스타트업이고,
주 고객은 중소기업 IT 담당자입니다.
이 맥락에서 답변해주세요.`}</pre>
</div>
</SlideOnly>

<DocOnly>

Claude.ai를 사용하는 경우, 대화의 첫 메시지에서 역할을 지정하면 됩니다:

```
당신은 10년 경력의 마케팅 전문가입니다.
우리 회사는 B2B SaaS 스타트업이고, 주 고객은 중소기업 IT 담당자입니다.
이 맥락에서 답변해주세요.
```

이렇게 대화를 시작하면, 이후 모든 답변이 마케팅 전문가 관점에서 제공됩니다.

## API에서 역할 설정하기 (개발자용)

```python
import anthropic

client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-opus-4-6",
    max_tokens=1024,
    system="당신은 Python 전문 코딩 어시스턴트입니다.",
    messages=[
        {"role": "user", "content": "딕셔너리 리스트를 키별로 정렬하려면 어떻게 하나요?"}
    ],
)
print(message.content)
```

### 역할 없이 vs 역할 부여 후

<Tabs items={['❌ 역할 없이', '✅ 고양이 역할 부여']}>


Claude: 스케이트보드에 대한 일반적이고 객관적인 답변

  <Tabs.Tab>

```

```

Claude: 고양이 관점의 재미있는 답변 (예: "시끄러운 바퀴 달린 것은 낮잠을 방해하니까 별로야. 냐옹.")

```
진수는 은지를 바라보고 있습니다. 은지는 현우를 바라보고
있습니다. 진수는 결혼했고, 현우는 결혼하지 않았습니다.
은지의 결혼 여부는 모릅니다.
결혼한 사람이 결혼하지 않은 사람을 바라보고 있나요?
```

<Callout emoji="💡">
역할 없이 물으면 Claude는 "알 수 없다"고 답할 수 있지만, **논리 봇 역할**을 부여하면 경우의 수를 체계적으로 분석하여 <strong>"예"</strong>라는 정답을 찾아냅니다.
</Callout>
