# Claude에게 역할 부여하기

<OriginalDocLink href="https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/claude-prompting-best-practices#give-claude-a-role">공식문서에서 보기</OriginalDocLink>

시스템 프롬프트에서 역할을 설정하면 Claude의 동작과 톤을 사용 사례에 맞게 집중시킵니다. **한 문장만으로도 차이가 납니다.**

## Claude.ai에서 역할 설정하기

Claude.ai를 사용하는 경우, 대화의 첫 메시지에서 역할을 지정하면 됩니다:

```
당신은 10년 경력의 마케팅 전문가입니다.
우리 회사는 B2B SaaS 스타트업이고, 주 고객은 중소기업 IT 담당자입니다.
이 맥락에서 답변해주세요.
```

이렇게 대화를 시작하면, 이후 모든 답변이 마케팅 전문가 관점에서 제공됩니다.

<Callout emoji="⚙️">
아래는 Claude API를 사용하는 개발자를 위한 코드 예시입니다. Claude.ai를 사용하는 직장인은 위의 방법을 참고하세요.
</Callout>

## API에서 역할 설정하기 (개발자용)

```python
import anthropic

client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-opus-4-6",
    max_tokens=1024,
    system="당신은 Python 전문 코딩 어시스턴트입니다.",
    messages=[
        {"role": "user", "content": "딕셔너리 리스트를 키별로 정렬하려면 어떻게 하나요?"}
    ],
)
print(message.content)
```

## 실무에서 이렇게 쓰세요

역할 부여는 **전문 분야의 관점이 필요한 업무**에서 결과 품질을 극적으로 높입니다.

/>

<Callout emoji="📊">
**역할 부여가 작동하는 3가지 레벨**: (1) **용어 보정** — "시니어 계약 변호사"는 법률 용어와 인용 인식을 활성화 (2) **암묵적 제약 설정** — "정중한 CS 에이전트"는 톤, 범위, 에스컬레이션 행동을 일일이 규칙으로 쓰지 않아도 함축 (3) **다중 에이전트 전문화** — 오케스트레이터/분석가/리뷰어로 역할을 분리하면 인간 팀의 전문 분업을 AI 파이프라인에 그대로 적용 가능.
</Callout>

## Anthropic 튜토리얼 예시

### 역할 없이 vs 역할 부여 후

<Tabs items={['❌ 역할 없이', '✅ 고양이 역할 부여']}>
  <Tabs.Tab>
```
한 문장으로, 스케이트보드에 대해 어떻게 생각해?
```

Claude: 스케이트보드에 대한 일반적이고 객관적인 답변
  </Tabs.Tab>
  <Tabs.Tab>
시스템 프롬프트: `당신은 고양이입니다.`

```
한 문장으로, 스케이트보드에 대해 어떻게 생각해?
```

Claude: 고양이 관점의 재미있는 답변 (예: "시끄러운 바퀴 달린 것은 낮잠을 방해하니까 별로야. 냐옹.")
  </Tabs.Tab>
</Tabs>

### 논리 문제 — 역할이 정확도를 높이는 경우

시스템 프롬프트: `당신은 복잡한 논리 문제를 해결하는 논리 봇입니다.`

```
진수는 은지를 바라보고 있습니다. 은지는 현우를 바라보고
있습니다. 진수는 결혼했고, 현우는 결혼하지 않았습니다.
은지의 결혼 여부는 모릅니다.
결혼한 사람이 결혼하지 않은 사람을 바라보고 있나요?
```

<Callout emoji="💡">
역할 없이 물으면 Claude는 "알 수 없다"고 답할 수 있지만, **논리 봇 역할**을 부여하면 경우의 수를 체계적으로 분석하여 **"예"**라는 정답을 찾아냅니다.
</Callout>

## 연습문제

### 문제: 수학 검증 역할

아래 방정식 풀이에는 오류가 있습니다. Claude가 **정확하게 오류를 찾도록** 시스템 프롬프트를 작성하세요.

```
2x - 3 = 9
2x = 6
x = 3
```

<Quiz
  hint="Claude에게 '수학 채점 보조' 같은 전문 역할을 부여하면 각 단계를 꼼꼼히 검증합니다"
  answer={<>
    <p>시스템 프롬프트:</p>

    <p>풀이 오류: <code>2x - 3 = 9</code> → <code>2x = 12</code> (6이 아님) → <code>x = 6</code> (3이 아님)</p>

>
Claude가 위 방정식의 오류를 정확하게 찾도록 하는 **시스템 프롬프트**를 작성해보세요.
</Quiz>

## 정리

오늘 배운 핵심을 정리합니다.

| # | 배운 것 | 핵심 |
|---|---------|------|
| 1 | **역할 = 한 문장의 맥락 폭탄** | "시니어 계약 변호사"라는 한 문장이 용어, 톤, 분석 깊이를 한꺼번에 설정 |
| 2 | **역할이 작동하는 3가지 레벨** | 용어 보정 → 암묵적 제약 설정 → 다중 에이전트 전문화 |
| 3 | **시스템 프롬프트 활용** | Claude.ai에서는 대화 시작에, API에서는 system 필드에 역할을 설정 |

<SlideOnly>
<div className="keyword-card summary">
<h3>오늘의 정리</h3>
<p>1. 역할 한 문장 = 용어 + 톤 + 분석 깊이 한꺼번에 설정</p>
<p>2. 용어 보정 → 암묵적 제약 → 다중 에이전트 전문화</p>
<p>3. 대화 시작에 역할을 설정하면 이후 모든 답변이 달라진다</p>
</div>
</SlideOnly>

<Callout emoji="💬">

**"역할을 부여하는 건 전문가를 고용하는 것과 같습니다."**

같은 질문이라도 누구에게 물어보느냐에 따라 답이 달라지듯, AI에게도 적절한 역할을 부여하면 전문가 수준의 답변을 얻을 수 있습니다.

</Callout>

### 오늘의 액션

다음에 AI한테 뭔가를 요청할 때, 대화 맨 첫 줄에 **역할 한 문장**을 추가해 보세요.

```
당신은 ___분야의 전문가입니다.
```
